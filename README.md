# StayFinder – PG & Hostel Search Management System

A web-based accommodation management platform that allows users to search and book PGs and hostels while enabling owners and administrators to manage listings.

Features include authentication, hostel search, booking system, role-based dashboards (Customer, Owner, Super Admin), and optimized data handling using Data Structures and Algorithms (DSA).

Start with `npm install` and `npm start`.

Technologies Used: HTML, CSS, JavaScript, React.js, Node.js, PostgreSQL.
